# מרתף היין שלי 🍷

אפליקציית ניהול אוסף יינות עם בינה מלאכותית. עובדת על iPhone, iPad, אנדרואיד וכל דפדפן.

## מה האפליקציה יודעת לעשות?
- 📸 לזהות בקבוקי יין מתמונה (אחד או כמה ביחד)
- 🧾 לסרוק חשבוניות קנייה ולחלץ את כל היינות
- ⌨️ הוספה ידנית או ייבוא מטבלת אקסל/CSV
- 🤖 להעשיר אוטומטית עם ציוני מבקרים, חלון שתייה, תווי טעימה ושילובי אוכל
- 🍽️ סומלייה חכם שממליץ מה לשתות לפי האוכל והאירוע
- 📊 סטטיסטיקות ותובנות על האוסף
- 📥 ייצוא לאקסל/CSV

## הוראות התקנה (15 דקות סה"כ)

### שלב 1: השג מפתח Anthropic API (5 דקות)

1. היכנס ל-**https://console.anthropic.com** והתחבר/הירשם
2. בתפריט בצד שמאל, לחץ על **API Keys**
3. לחץ **Create Key**, תן לו שם (למשל "wine-app"), לחץ Create
4. **העתק את המפתח מיד** (הוא מתחיל ב-`sk-ant-`) — לא תוכל לראות אותו שוב
5. לך לתפריט **Billing** והוסף לפחות $5 קרדיט. זה יספיק לכמאתיים זיהויים.

### שלב 2: צור חשבון GitHub והעלה את הקוד (3 דקות)

1. היכנס ל-**https://github.com** והירשם (חינם)
2. בפינה הימנית העליונה, לחץ על **+** ואז **New repository**
3. שם: `wine-cellar` (או כל שם אחר). השאר "Public", לחץ **Create repository**
4. בעמוד שנפתח, לחץ על הקישור **"uploading an existing file"** (באמצע העמוד)
5. **גרור את כל הקבצים** מהתיקייה (אחרי שחילצת את ה-ZIP) — חשוב: תגרור את הקבצים, לא את התיקייה. אם זה לא עובד, בחר את כל הקבצים ב-Finder (Cmd+A) וגרור ביחד.
6. גלול למטה ולחץ **Commit changes**

### שלב 3: פרוס ב-Vercel (3 דקות)

1. היכנס ל-**https://vercel.com** ולחץ **Sign Up**
2. בחר **Continue with GitHub** והרשה גישה
3. אחרי ההתחברות, לחץ **Add New** (כפתור שחור) ובחר **Project**
4. תראה רשימה של הפרויקטים שלך ב-GitHub. ליד `wine-cellar` לחץ **Import**
5. **חשוב מאוד — לפני שאתה לוחץ Deploy:**
   - גלול למטה לסעיף **Environment Variables**
   - בשדה **Key**: הקלד `ANTHROPIC_API_KEY` (בדיוק כך, באותיות גדולות עם קווים תחתונים)
   - בשדה **Value**: הדבק את המפתח שהעתקת בשלב 1
   - לחץ **Add**
6. עכשיו לחץ **Deploy**. תמתין דקה-שתיים.
7. כשתראה ✅ ירוק, לחץ על התמונה הקטנה של האתר או על הקישור — זה ה-URL שלך! משהו כמו `wine-cellar-xxx.vercel.app`

### שלב 4: הוסף לאייפון כאפליקציה (30 שניות)

1. פתח את ה-URL מהשלב הקודם **ב-Safari** (חשוב, לא Chrome)
2. לחץ על כפתור השיתוף (ריבוע עם חץ למעלה, באמצע הסרגל התחתון)
3. גלול למטה ובחר **Add to Home Screen**
4. תן שם (למשל "מרתף") ולחץ **Add**

✨ עכשיו יש לך אייקון יין במסך הבית של האייפון. לחיצה עליו תפתח את האפליקציה במסך מלא, בדיוק כמו אפליקציה רגילה.

## פתרון בעיות

**"שגיאה בניתוח: API 500" כשמנסה להעלות תמונה**
- כנראה שלא הגדרת את `ANTHROPIC_API_KEY` ב-Vercel. לך ל-Project Settings → Environment Variables ובדוק.
- אם הוספת אבל זה לא עובד: לך ל-Deployments, לחץ על הנקודות (...) ליד הפריסה האחרונה, ובחר **Redeploy**.

**"API 401" או "API 403"**
- המפתח שגוי או פג תוקפו. צור חדש ב-console.anthropic.com והחלף ב-Vercel.

**"API 429"**
- לא הוספת קרדיט בחשבון Anthropic. לך ל-Billing והוסף לפחות $5.

**הפריסה ב-Vercel נכשלה (Build error)**
- ודא שהעלית את **כל הקבצים** ל-GitHub (פתח את ה-repository ובדוק שיש שם package.json, vite.config.js, src/, api/, וכל השאר).
- אם חסר קובץ, פתח אותו מהמחשב והעלה אותו לבד דרך GitHub.

**האפליקציה נפתחת אבל ריקה / לבנה**
- פתח את ה-Console בדפדפן (Settings → Safari → Advanced → Web Inspector באייפון, או F12 במחשב) ותראה אם יש שגיאה.

- **Vercel**: חינם לחלוטין לשימוש אישי.

## פרטיות
- המפתח שלך שמור רק ב-Vercel ולא חשוף בדפדפן.
- האוסף שלך נשמר רק במכשיר שלך (localStorage), לא נשלח לשום שרת.
- תמונות נשלחות ל-Anthropic לזיהוי ולא נשמרות שם.
